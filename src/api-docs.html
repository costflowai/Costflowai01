📡 API Endpoints

POST /api/v1/estimate Get construction cost estimate

Calculate construction costs based on project parameters.

Request Parameters

Parameter Type Required Description
project_type string Required Type of construction project (residential, commercial, etc.)
square_footage number Required Total square footage of the project
location string Required ZIP code or city, state for location-based pricing
quality_level string Optional Quality tier: basic, standard, premium, luxury
timeline string Optional Project start date for seasonal pricing adjustments

Example Request

// Using fetch API const response = await fetch('https://api.costflowai.com/v1/estimate', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ 'project_type': 'residential', 'square_footage': 2500, 'location': '90210', 'quality_level': 'premium' }) }); const estimate = await response.json();

Example Response

{ "success": true, "estimate": { "total_cost": 625000, "cost_per_sqft": 250, "breakdown": { "materials": 281250, "labor": 218750, "permits": 31250, "contingency": 93750 }, "accuracy_range": { "min": 562500, "max": 687500 }, "confidence_score": 0.92, "location_factors": { "cost_index": 1.3, "labor_availability": "high" } }, "request_id": "est_1234567890" }
GET /api/v1/market-data Get real-time market intelligence

Retrieve current market conditions, pricing trends, and regional insights.

Query Parameters

Parameter Type Required Description
location string Required ZIP code or city, state
category string Optional materials, labor, permits, all (default)
period string Optional 1m, 3m, 6m, 1y for trend analysis

Example Response

{ "success": true, "market_data": { "location": "Los Angeles, CA", "cost_index": 127.5, "trends": { "materials": { "current_index": 132.1, "change_3m": 2.3, "forecast_6m": -1.2 }, "labor": { "availability": "moderate", "rate_change": 4.1 } } } }
POST /api/v1/photo-analysis AI photo-to-estimate

Upload photos to get AI-powered construction estimates using computer vision.

Request Parameters

Parameter Type Required Description
images array Required Base64 encoded images or image URLs
analysis_type string Optional room_analysis, damage_assessment, renovation
location string Optional For location-based pricing

Example Response

{ "success": true, "analysis": { "detected_features": [ "granite_countertops", "hardwood_flooring", "stainless_appliances" ], "room_type": "kitchen", "estimated_sqft": 180, "cost_estimate": { "total": 45000, "breakdown": { /* ... */ } }, "confidence": 0.87 } }
